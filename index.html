Command Line Shell: Implemented a Unix shell with the same abilities as bash. Used C system calls (execvp, dup2, pipe) to enable scripting as well as I/O redirection.
Memory Allocator: Developed a custom memory allocator using a Linked List data structure. Allowed for custom algorithms for how blocks of memories would be split when memory is requested.
